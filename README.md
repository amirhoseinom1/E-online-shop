# E-online-shop

[English]

A simple e-commerce website built with Django.  
Hope you find this project interesting and useful! 🚀

## Features
- Product list with pagination
- Product detail page
- User registration, login, and logout
- Comment system with admin approval
- Admin panel for managing products and comments

## Installation
1. Clone the repository
2. Create a virtual environment
3. Install requirements: `pip install -r requirements.txt`
4. Run migrations: `python manage.py migrate` (migration files are included)
5. Create a superuser: `python manage.py createsuperuser`
6. Run the server: `python manage.py runserver`

# فروشگاه اینترنتی

یک فروشگاه اینترنتی ساده ساخته شده با جنگو.  
امیدوارم این پروژه برای شما جذاب و مفید باشد! 🚀

## قابلیت‌ها
- لیست محصولات با صفحه‌بندی
- صفحه جزئیات محصول
- ثبت‌نام، ورود و خروج کاربران
- سیستم کامنت با تایید ادمین
- پنل مدیریت برای مدیریت محصولات و کامنت‌ها

## نصب و راه‌اندازی
1. ریپازیتوری را کلون کنید
2. یک محیط مجازی بسازید
3. نصب پکیج‌ها: `pip install -r requirements.txt`
4. اجرای مایگریشن‌ : `python manage.py migrate` 
5. ساخت کاربر ادمین: `python manage.py createsuperuser`
6. اجرای سرور: `python manage.py runserver`
